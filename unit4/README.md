# Unit 4: Maven - Build Automation and Project Management

This unit covers Apache Maven, a powerful build automation tool and project management framework for Java projects.

## Topics Covered

1. **Maven Introduction** - What is Maven and why use it
2. **POM Structure** - Understanding the Project Object Model
3. **Maven Lifecycle** - Build phases and goals
4. **Dependency Management** - Managing project dependencies
5. **Maven Plugins** - Extending Maven functionality
6. **Maven Docker Integration** - Containerizing Maven projects

## Learning Outcomes

- Understand Maven architecture and principles
- Create and configure Maven projects
- Manage project dependencies effectively
- Use Maven lifecycle for build automation
- Integrate Maven with Docker
- Build reproducible and portable Java applications
