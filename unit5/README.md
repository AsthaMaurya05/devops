# Unit 5: GitHub Actions CI/CD

This unit covers GitHub Actions, a powerful CI/CD platform for automating workflows directly in your GitHub repository.

## Topics Covered

1. **Introduction to GitHub Actions** - Basics and concepts
2. **Workflow Triggers** - Events that trigger workflows
3. **Jobs, Steps, and Actions** - Building blocks of workflows
4. **Matrix Strategies and Caching** - Advanced workflow patterns
5. **Docker and GitHub Actions** - Containerization in CI/CD
6. **Java/Maven CI** - Building Java applications with Maven


## Learning Objectives

- Understand GitHub Actions architecture and components
- Create and manage workflows for CI/CD pipelines
- Integrate Docker with GitHub Actions
- Build and push container images automatically
- Secure sensitive data using GitHub Secrets
- Implement CI/CD for Java/Maven projects
