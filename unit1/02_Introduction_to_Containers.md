# 📌 1. What are Containers?

A **container** is a lightweight package that contains:

- Application code
- Runtime environment
- System libraries
- Dependencies
- Configuration files

This allows the application to run **consistently on any system**.

> 💡 **Simple Definition**  
> A container is a portable environment that packages an application and everything it needs to run.
